# 🤖 Cognitia AI Chat App

## 📌 Project Overview
Cognitia is a full-stack AI-powered chat application that allows users to interact with an AI assistant in a simple conversational format. The system sends user questions to a backend API powered by Groq LLM and returns intelligent responses in real time.

---

## 🚀 Features
- Single question → single AI response
- Real-time chat interface
- Stores Q&A in MongoDB Atlas
- Clean UI
- Fully deployed on cloud

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- JavaScript

### Backend
- Node.js
- Express.js
- Groq API (Llama 3.1 8B Instant)

### Database
- MongoDB Atlas

### Deployment
- Frontend: Vercel  
- Backend: Render
